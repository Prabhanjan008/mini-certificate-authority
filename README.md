# 🔐 Mini Certificate Authority (Mini CA)

## 📌 Overview
This project demonstrates the working of Public Key Infrastructure (PKI) by creating a Mini Certificate Authority using OpenSSL.

It includes generating and signing digital certificates and deploying a secure HTTPS web application using Flask.

---

## ⚙️ Technologies Used
- OpenSSL
- Python (Flask)
- Web Browser (Chrome)
- Windows OS

---

## 🚀 Features
- Create self-signed Certificate Authority
- Generate server certificates
- Sign certificates using CA
- Enable HTTPS communication
- Demonstrate browser certificate warning

---

## 🧠 Concepts Covered
- Public Key Infrastructure (PKI)
- SSL/TLS Encryption
- Digital Certificates
- Trust Chain Validation

---

## ▶️ How to Run

1. Generate CA using OpenSSL
2. Create server certificate
3. Sign certificate using CA
4. Run Flask server with HTTPS
